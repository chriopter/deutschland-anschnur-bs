# Deutschland Anschnur BS

Ein lokalisiertes Bogenleinux-Derivat mit `muschel`, einer erlaubnisbasierten Kleinmuschel. Neueste DIN-Anlage: [Herunterladen](https://github.com/chriopter/deutschland-anschnur-bs/releases/latest/download/deutschland-anschnur-bs-aktuell.iso).

Einige angelsächsische Restbegriffe verbleiben aufgrund technischer Zuständigkeitsbindung bis zur späteren Verdeutschungsverfügung im Bestand.

## Befehle

### Aktualisieren

Ein eingerichtetes Deutschland Anschnur BS wird über `hapsmann erneuern` aktualisiert; die zugehörigen Anschnur-Pakete werden in der [Deutschland-Anschnur-Paketablage](https://chriopter.github.io/deutschland-anschnur-bs/x86_64/) gepflegt.

### Muschel

| Befehl | Dienstliche Wirkung |
| --- | --- |
| `hilfe` | erteilt eine knappe Befehlsauskunft |
| `hapsmann` | bearbeitet Paketangelegenheiten |
| `verlassen` | beendet den Muschelvorgang ordnungsgemäß |

Alle übrigen Eingaben werden amtlich beanstandet. Die Muschel teilt dabei mit, ob der angelsächsische Fremdbefehl im System vorhanden, aber noch nicht freigegeben ist.

### Hapsmann

| Befehl | Dienstliche Wirkung |
| --- | --- |
| `hapsmann erneuern` | erneuert Paketbestand und System |
| `hapsmann suchen <Begriff>` | sucht verfügbare Pakete |
| `hapsmann zeigen <Paket>` | zeigt ein eingerichtetes Paket |
| `hapsmann fernzeigen <Paket>` | zeigt ein verfügbares Paket |
| `hapsmann installieren <Paket...>` | richtet Pakete ein |
| `hapsmann entfernen <Paket...>` | entfernt Pakete samt Folgebestand |
| `hapsmann bestand` | listet eingerichtete Pakete |
| `hapsmann aufräumen` | räumt den Paketzwischenspeicher auf |

Die Muschel wird als Paket `anschnur-muschel` bereitgestellt. Gewöhnliche Bogenleinux-Erneuerung bezieht dadurch zugleich die neueste Muschel aus der Deutschland-Anschnur-Paketablage.
