# Deutschland Anschnur Muschel

Eine tabellengeführte Muschel zur Verdeutschung von Befehlen auf bestehenden Einix-/Leinux-Systemen.

Einige angelsächsische Restbegriffe verbleiben aufgrund technischer Zuständigkeitsbindung bis zur späteren Verdeutschungsverfügung im Bestand.

## Befehle

### Einrichtung

```bash
git clone https://github.com/chriopter/deutschland-anschnur-muschel.git ~/.deutschland-anschnur-muschel
echo 'alias muschel="$HOME/.deutschland-anschnur-muschel/muschel"' >> ~/.bashrc
. ~/.bashrc
```

### Erneuerung

```bash
cd ~/.deutschland-anschnur-muschel
git pull --ff-only
```

### Start

```bash
muschel
muschel verzeichnis liste .
```

### Muschel

| Befehl | Wirkung |
| --- | --- |
| `hilfe` | zeigt die Befehlsauskunft |
| `verlassen` | schließt die Muschel |
| `muschel neu laden` | lädt Befehlsakten neu |

### Grundbestand

| Befehl | Wirkung |
| --- | --- |
| `verzeichnis ort` | zeigt den derzeitigen Ort |
| `verzeichnis liste [Pfad]` | zeigt einen Verzeichnisbestand |
| `verzeichnis wechsel <Pfad>` | wechselt den derzeitigen Ort |
| `datei lesen <Pfad>` | liest eine Datei |
| `datei zeigen <Pfad>` | zeigt Dateiangaben |
| `netz stand` | zeigt Netzanschriften |
| `netz weg` | zeigt Netzwege |
| `netz pruefen <Ziel>` | prüft die Erreichbarkeit |
| `dienst liste` | zeigt laufende Dienste |
| `dienst stand <Name>` | zeigt einen Dienststand |
| `system stand` | zeigt Systemstand |
| `system kennung` | zeigt Betriebssystemkennung |

### Weitere Befehlsakten

| Befehl | Wirkung |
| --- | --- |
| `hapsmann erneuern` | erneuert Paketbestand und System, sofern Hapsmann-Bestand vorhanden ist |
| `hafener zusammensetz hoch --dämonisiere` | übersetzt einen Hafener-Zusammensetzvorgang |
