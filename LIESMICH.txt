# Deutschland Anschnur Muschel

Eine tabellengeführte Muschel zur Verdeutschung von Befehlen auf bestehenden Einix-/Leinux-Systemen.

Einige angelsächsische Restbegriffe verbleiben aufgrund technischer Zuständigkeitsbindung bis zur späteren Verdeutschungsverfügung im Bestand.

## Einrichtung

```bash
git clone https://github.com/chriopter/deutschland-anschnur-muschel.git ~/.deutschland-anschnur-muschel
echo 'alias muschel="$HOME/.deutschland-anschnur-muschel/bin/muschel"' >> ~/.bashrc
. ~/.bashrc
```

Die Muschel meldet verfügbare Erneuerungen beim Start.

**Muschel**

```bash
muschel
muschel verzeichnis liste .
```

### Muschel

| Befehl | Wirkung |
| --- | --- |
| `hilfe` | zeigt diese dienstliche Auskunft |
| `verlassen` | schließt die Muschel ordnungsgemäß |
| `muschel neu laden` | lädt die Befehlsakten neu |

### Grundbestand

| Befehl | Wirkung |
| --- | --- |
| `verzeichnis ort` | zeigt den derzeitigen Ort |
| `verzeichnis liste` | zeigt einen Verzeichnisbestand |
| `verzeichnis wechsel` | wechselt den derzeitigen Ort |
| `datei lesen` | liest eine Datei |
| `datei zeigen` | zeigt Dateiangaben |
| `netz stand` | zeigt Netzanschriften |
| `netz weg` | zeigt Netzwege |
| `netz pruefen` | prüft die Erreichbarkeit |
| `netz prüfen` | prüft die Erreichbarkeit |
| `dienst liste` | zeigt laufende Dienste |
| `dienst stand` | zeigt einen Dienststand |
| `system stand` | zeigt Systemstand |
| `system kennung` | zeigt Betriebssystemkennung |
| `system neustarten` | startet neu |
| `system ausschalten` | schaltet aus |

| `hapsmann suchen` | sucht verfügbare Pakete |
| `hapsmann zeigen` | zeigt ein eingerichtetes Paket |
| `hapsmann fernzeigen` | zeigt ein verfügbares Paket |
| `hapsmann installieren` | richtet Pakete ein |
| `hapsmann entfernen` | entfernt Pakete samt Folgebestand |
| `hapsmann bestand` | listet eingerichtete Pakete |
| `hapsmann aufraeumen` | räumt den Paketzwischenspeicher auf |
| `hapsmann aufräumen` | räumt den Paketzwischenspeicher auf |
